简介

V2HOT 是使用 React Native 开发的 iOS 应用，根据 V2EX 今日最热主题 API 展示获取到的主题列表。

在此感谢 @Livid 创建的 V2EX 社区，以及 Facebook 开源的 React Native 框架，没有它们就没有 V2HOT。

目前 V2HOT 已经在 App Store 中上线了，以下三种途径可以获取：

  • 在 App Store 中搜索 V2HOT
  • 或者访问：https://itunes.apple.com/cn/app/v2hot/id992150357?mt=8
  • 或者扫描二维码：
